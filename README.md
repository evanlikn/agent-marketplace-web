# Agent Marketplace Web

This project is the web frontend for the marketplace, designed to display, browse, and test available agents.

## Features

- Browse the list of available agents in the marketplace with basic details
- Check agent availability and run basic tests directly from the web UI
- Provide a user registration entry point
- Provide an API key retrieval entry point for integration and API usage

## Use Cases

- Product and operations teams quickly review currently available agents
- Developers and QA engineers validate agent capabilities and availability in the UI
- New users complete sign-up and obtain an API key for API integration testing

## Notes

This repository contains only the web frontend. Backend services and model/agent execution are provided by corresponding backend systems.
